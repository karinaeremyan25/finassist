---
description: Правила работы с денежными суммами в FinAssist
globs: ["**/*.ts"]
---
- Все суммы хранятся в **копейках (BIGINT)**. Никогда не использовать float для денег
- Конвертация только через `src/utils/money.ts`:
  - `rubles(kopecks: bigint): string` — форматирование для вывода
  - `toKopecks(input: string | number): bigint` — парсинг ввода пользователя
- В БД: колонки типа `BIGINT` с названием `amount_kopecks` или `amount`
- В TypeScript: тип `bigint` для сумм, не `number`
- При парсинге ввода от пользователя (например, "1500.50 руб") — парсить через `utils/money.ts`
- При отображении: всегда форматировать как "1 500,50 ₽"
- Валютные суммы: хранить и оригинальную сумму в валюте, и рублёвый эквивалент (по fx_rates)
- НЕТ: `parseFloat`, `/100`, `* 100` без использования utils/money.ts
