- Пользовательский ввод ВСЕГДА оборачивать: `<user_input>...</user_input>`
- Confidence thresholds (строго):
  - ≥ 0.85 → сразу карточка подтверждения, без вопросов
  - 0.7–0.84 → один уточняющий вопрос по наименее уверенному полю
  - < 0.7 → уточнения по всем неуверенным полям последовательно
- Ответ Claude — структурированный JSON с confidence per field
- Температура: 0 (детерминированность классификации)
- При недоступности Anthropic API → fallback на ручной ввод с инструкцией:
  `сумма/юрлицо/направление/категория/описание`
- Полный текст транзакций НЕ логировать (только metadata: user_id, category, confidence)
- Retry для Anthropic API: 3 попытки, 1s/3s/9s; при 429 — задержка 30s
- Системный промпт: загружать entity и direction из БД динамически (не хардкодить)
- Модель: `claude-sonnet-4-6` (из config.ts, не хардкодить в сервисе)
